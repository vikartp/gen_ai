# Agentic AI

Experimental agentic AI implementations demonstrating progressive agent patterns — from basic single-tool agents to multi-agent orchestration with LangGraph.

## Prerequisites

- **Python** 3.12 or higher
- **uv** (recommended)
- **API Key**: OpenAI or OpenRouter

## Setup

### 1. Create Virtual Environment

```bash
uv venv
```

### 2. Activate Virtual Environment

- **Windows:** `.venv\Scripts\activate`
- **Linux/Mac:** `source .venv/bin/activate`

### 3. Install Dependencies

**Using `pyproject.toml` (preferred):**

```bash
uv sync
```

**Or install as editable package:**

```bash
uv pip install -e .
```

**Or use the setup script:**

```bash
bash agentic-setup.sh
```

### 4. Configure Environment Variables

```bash
cp .env.example .env
```

Edit `.env` and replace `your-api-key-here` with your actual API key.

**Windows (PowerShell):**

```powershell
Copy-Item .env.example .env
```

> You can use either OpenAI (`sk-...`) or OpenRouter (`sk-or-v1-...`) API keys.

python agents/agent_1_basic.py

python agents/agent_3_memory.py

- `langgraph` — Multi-agent orchestration
- `langchainhub` — Prompt templates
- `duckduckgo-search` / `ddgs` — Web search tools
- `yfinance` — Financial data
- `python-dotenv` — Environment variable management

## Troubleshooting

### ModuleNotFoundError

Ensure virtual environment is activated and dependencies are installed:

```bash
.venv\Scripts\activate
uv pip install <missing-package>
```

### OpenAI API rate limits

Consider using OpenRouter for access to multiple LLM providers with better rate limits.
